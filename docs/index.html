hibiki

Android 日语沉浸式阅读器

面向日语学习者的 EPUB 阅读、Yomitan 词典查询、AnkiDroid 制卡与 Sasayaki 有声书同步工具。 在阅读、查词、摘句、制卡和跟读之间减少切换，让学习流保持连贯。

GitHub README API Docs
EPUB Reading

WebView 内嵌 ttu Ebook Reader，支持点按查词、主题、字体、统计、连续滚动与分页模式。

Dictionary

导入 Yomitan 格式词典，支持音调、词频、多词典并行查询、搜索历史与 Ve 词形还原。

Anki Export

通过 AnkiDroid 一键制卡，自动填充上下文句子，支持录音、截图裁剪和多导出配置。

Sasayaki

将 SRT、LRC、VTT、ASS 字幕对齐 EPUB 正文，提供跟读高亮和音频同步翻页。
